    My recommended order if you wanted to chase Ruff parity

    1. Compare — biggest 1-wrapper-to-rules ratio. Unlocks four E/F default rules immediately.
    2. Import + ImportFrom + alias — needed for any plugin that reasons about imports (the most common "real" rule category — flake8-bugbear, isort, etc.).
    3. FunctionDef + ClassDef + Assign — unlocks rules that reason about definitions (mutable defaults, naming).
    4. ExceptHandler for E722 (one line of rule code, very common finding).
    5. Symbol table (the "natural next milestone" bullet in MVP.md). Once you have this, F401/F811/F821/F841 are each a few lines.
    6. Everything else as rules demand.
